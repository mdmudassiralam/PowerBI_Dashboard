# 📊 PhonePe Dashboard

## Overview
The **PhonePe Dashboard** is an interactive Power BI project designed to analyze and visualize digital payment transactions and user activity data. The dashboard provides meaningful insights into transaction trends, user growth, and regional performance through dynamic visualizations.

## Features
- Interactive and user-friendly dashboard.
- Analysis of transaction volume and transaction value.
- State-wise and region-wise performance comparison.
- User activity and growth analysis.
- Trend analysis using charts and KPIs.
- Filters and slicers for customized data exploration.

## Tools and Technologies
- **Power BI**
- **Microsoft Excel / CSV Dataset**
- **Data Cleaning and Transformation using Power Query**
- **DAX (Data Analysis Expressions)**
- **Dashboarding design using figma**

  ## Dashboard Components
- KPI Cards
- Bar Charts
- Line Charts
- Pie Charts
- Maps and Geographic Visualizations
- Slicers and Filters

  ## Data Source
- **Dataset:** PhonePe Transactions Dataset (2024)
- **Source:** Kaggle
- **Data Includes:**
  - Transaction count
  - Transaction amount
  - State-wise transaction details
  - User activity data
  - Category-wise transaction analysis

  ## Features

* 📊 Interactive Power BI dashboard for analyzing PhonePe transactions in 2024.
* 📅 Month-wise transaction analysis to identify trends throughout the year.
* 💳 Category-wise transaction insights, including loans and other transaction types.
* ✅ Transaction status analysis to monitor Successful, Pending, and Failed transactions.
* 👥 Generation-wise analysis to identify which age group contributes the most transactions.
* 📈 Key Performance Indicators (KPIs) for total transaction count and overall performance.
* 🎛️ Interactive filters and slicers for customized data exploration.
* ⚡ Data cleaning and transformation using Power Query.
* 🧮 DAX measures and calculations for generating meaningful business insights.
* 📉 Visualizations using bar charts, line charts, pie charts, and KPI cards.
